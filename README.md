# 🗺️ خارطة طريق أساسيات علوم الحاسب | Computer Science Fundamentals Roadmap

خارطة طريق برمجية، مُنظمة ومُرتبة منطقياً لمساعدتك في بناء أساس أكاديمي وتطبيقي صلب في علوم الحاسب (Computer Science). تم تقسيم المحتوى إلى مستويات متدرجة لتسهيل عملية المذاكرة والمتابعة.

---

## 📑 تفاصيل الخارطة المنهجية

### 🟢 المستوى الأول: حجر الأساس (The Baseline)

#### 1. أساسيات البرمجة (Programming Basics)
- [ ] **المتغيرات وأنواع البيانات** `Variables & Data Types`
- [ ] **عمليات الإدخال والإخراج** `Input / Output`
- [ ] **العمليات الأساسية** `Operators` (الحسابية، المنطقية، الشرطية)
- [ ] **الجمل الشرطية** `Conditions` (if, switch)
- [ ] **الحلقات التكرارية** `Loops` (for, while, do while)
- [ ] **الدوال** `Functions` (المعاملات، الإرجاع، والنطاق scope)
- [ ] **المصفوفات** `Arrays`

#### 2. حل المشكلات (Problem Solving)
- [ ] **تتبع الكود** `Tracing` لفهم مسار التنفيذ.
- [ ] **الكود الزائف والمخططات الانسيابية** `Pseudocode & Flowcharts`.
- [ ] **مهارة تفكيك المشاكل** وتقسيمها إلى خطوات عمل صغيرة.
- [ ] **تطبيق عملي** وحل مسائل سهلة على منصات مثل (HackerRank / Codeforces).

---

### 🟡 المستوى الثاني: البرمجة كائنية التوجه (Object-Oriented Programming)

#### البرمجة كائنية التوجه (OOP)
- [ ] **مفهوم الفئات والكائنات** `Class / Object`
- [ ] **دوال البناء والهدم** `Constructors / Destructors`
- [ ] **التغليف** `Encapsulation` (Getters / Setters)
- [ ] **الوراثة** `Inheritance`
- [ ] **تعدد الأشكال** `Polymorphism` (Overloading / Overriding)
- [ ] **التجريد** `Abstraction`
- [ ] **مفاهيم خاصة** `Advanced` (Static, Friend, This pointer)

---

### 🟠 المستوى الثالث: الهياكل والعمليات (Data Structures & Algorithms)

#### 1. بنيات البيانات (Data Structures)
- [ ] **الهياكل الخطية** `Stack / Queue`
- [ ] **القوائم المتصلة** `Linked List` (أحادية، ثنائية، دائرية)
- [ ] **مفهوم التكرار والعمق** `Recursion`
- [ ] **الأشجار** `Trees` (Binary Tree / Binary Search Tree)
- [ ] **طرق جولات الأشجار** `Traversals`
- [ ] **أساسيات المخططات** `Graph Basics` (BFS / DFS)
- [ ] **جداول الهاش** `Hash Tables` (المفهوم العام)

#### 2. الخوارزميات (Algorithms)
- [ ] **تحليل تعقيد الكود** `Complexity Analysis` (Time & Space Complexity)
- [ ] **خوارزميات الترتيب (Sorting):**
  - [ ] Bubble & Selection
  - [ ] Insertion Sort
  - [ ] Merge Sort & Quick Sort
- [ ] **خوارزميات البحث (Searching):**
  - [ ] Linear Search
  - [ ] Binary Search
- [ ] **تطبيقات متقدمة:**
  - [ ] تطبيقات على الـ Recursion
  - [ ] الخوارزميات الجشعة `Greedy Algorithms`
  - [ ] البرمجة الديناميكية `Dynamic Programming`

---

### 🔵 المستوى الرابع: هندسة الكود المنطقي (Logic & Architecture)

#### 1. التفكير المنطقي (Logic Design)
- [ ] **تحليل المشكلة** بعمق وتحديد المداخل والمخارج بدقة.
- [ ] **استراتيجية فرق تسد** `Divide & Conquer Strategy`.

#### 2. أنماط التصميم (Design Patterns Basics)
- [ ] **نمط السنجلتون** `Singleton Pattern`
- [ ] **نمط المصنع** `Factory Pattern`
- [ ] **نمط المراقب** `Observer Pattern`
